
-- SUMMARY --

Context Request Path is a Context Condition Module
that implements the Request Path context condition.
It uses the same configuration style as Context Path
but it uses the actual request path in stead of the internal node path.
This means that checks will be done on the page alias, or subpaths.

-- REQUIREMENTS --

Context

-- INSTALLATION --

Standard Drupal module installation, see http://drupal.org/node/70151.
Downloading of the Context module
Enabling of the Context module

-- CONFIGURATION --

Use the Context configuration under Structure
to define your request context paths
